# AP2 Interop Fixture Gap

As of 2026-06-15, this repository does not include a known-good AP2 mandate
fixture generated by a non-Steelyard implementation.

The v0.5 conformance harness covers AP2 with generated Steelyard fixtures,
parser checks, merchant authorization round trips, nonce replay checks, and the
coffee-shop AP2 smoke test. A future fixture from an independent AP2 wallet,
PSP, or reference implementation should be added in this directory with:

- the source implementation and version
- the checkout response used for signing
- the `ap2.checkout_mandate` presentation
- the payment mandate presentation
- the expected verification result
